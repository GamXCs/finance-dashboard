Finance Dashboard

A Python-based finance dashboard that helps users model long-term financial growth through savings, investing, and retirement planning.

Project Goals

This project is being built to:

* Practice Python programming
* Learn collaborative software development with Git and GitHub
* Gain experience organizing a larger codebase
* Apply data analysis using Pandas and NumPy
* Create data visualizations with Matplotlib
* Build a foundation for future data science and machine learning projects

Planned Features

Version 1

* Calculate investment growth over time
* Track monthly contributions
* Display total contributions vs. investment gains
* Generate growth charts

Future Versions

* High-yield savings account (HYSA) calculator
* Inflation-adjusted projections
* Multiple investment scenario comparisons
* Portfolio allocation simulator
* Retirement planning tools
* CSV import and export
* Interactive dashboard

Technologies
